## ğŸŒŸ About The Project

**AgroVia** is a cutting-edge platform designed to transform the food supply chain by providing complete transparency and quality assurance from farm to retail. In an era where food safety and sustainability are paramount, AgroVia bridges the gap between farmers, distributors, retailers, and consumers.

### Why AgroVia?

Modern agriculture faces critical challenges in maintaining food quality, reducing waste, and ensuring consumer trust. AgroVia addresses these challenges with a comprehensive tracking and analytics platform that benefits every stakeholder in the supply chain.

## ğŸš€ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

```bash
Node.js >= 18.0.0
npm >= 9.0.0 or yarn >= 1.22.0
```

### Installation

**1. Clone the repository**

```bash
git clone https://github.com/AnujPisal12/AgroVia1.0.git
cd AgroVia1.0
```

**2. Install dependencies**

```bash
npm install
# or
yarn install
```

**3. Start the development server**

```bash
npm run dev
# or
yarn dev
```

**4. Open your browser**

Navigate to `http://localhost:5173` to see AgroVia in action! ğŸ‰

### Build Commands

```bash
# Development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint code
npm run lint
```
